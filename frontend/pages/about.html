Nosso Propósito

Este projeto foi desenvolvido com o objetivo de facilitar o acesso à informação sobre unidades de saúde para toda a população, especialmente para idosos que muitas vezes enfrentam dificuldades ao buscar atendimento médico.

A plataforma permite visualizar hospitais disponíveis, os serviços oferecidos e a localização no Google Maps. Tudo isso com uma interface limpa, acessível e de fácil navegação.
